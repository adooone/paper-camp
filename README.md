# Paper Camp

Local-first, single-project planning dashboard with a retro-analog aesthetic.

## Quick Start

```bash
# Start the dev server
bun run dev
```

Open `http://localhost:3333` to access the dashboard.

## Pages

- **Plans** — Browse and manage plans from `papercamp/plans.md`
- **Focus** — Isolate a plan's tasks for focused work
- **Settings** — Project configuration

## How it works

Paper Camp reads and writes markdown files in a `papercamp/` directory at your project root. All data stays local — no server, no cloud.
